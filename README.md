# webservice

## Program language
Java 1.8 or later

## Framework
Spring boot

## Dependency
Maven

## How to test it
run command like this in linux platform
```shell
./mvnw test
```

run command like this in windows platform
```shell
./mvnw.cmd test
```

## How to deploy it locally
run command like this in linux platform
```shell
./mvnw spring-boot:run
```

run command like this in windows platform
```shell
./mvnw.cmd spring-boot:run
```
